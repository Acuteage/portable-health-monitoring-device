# Portable Health Monitoring Device

## 📌 Overview
This project presents a portable health monitoring system designed to track important physiological parameters in real time. It is a low-cost, wearable solution aimed at improving early detection of cardiovascular issues.

## ❗ Problem Statement
Cardiovascular diseases are one of the leading causes of death in India. Many patients lack access to continuous monitoring, leading to delayed diagnosis and treatment. This project aims to provide an affordable and real-time health monitoring solution.

## 💡 Solution
The system monitors:
- ECG (Electrocardiogram)
- Heart Rate (BPM)
- Blood Oxygen Level (SpO₂)

Data is collected using sensors and transmitted wirelessly via Bluetooth for real-time monitoring.

## ⚙️ Components Used
- Arduino Nano
- AD8232 ECG Sensor
- MAX30102 Sensor
- HC-05 Bluetooth Module
- TP4056 Charging Module
- Li-Po Battery

## 🔄 Working Principle
1. ECG electrodes capture heart signals
2. AD8232 processes ECG signals
3. MAX30102 measures BPM and SpO₂
4. Arduino processes all data
5. Bluetooth module sends data to mobile/PC
6. Data is displayed in real-time

## 📊 Features
- Real-time health monitoring
- Wireless data transmission
- Portable and wearable design
- Low-cost implementation
