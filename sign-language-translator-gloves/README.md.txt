# 🤟 Sign Language Translator Gloves

## 📌 Project Overview
This project presents a wearable smart glove system designed to translate Indian Sign Language (ISL) gestures into readable text in real time.

It aims to bridge the communication gap between hearing/speech-impaired individuals and the general public using an affordable and portable solution.

---

## ❗ Problem Statement
Most existing sign language translation systems:
- Support only limited gestures  
- Focus on foreign sign languages (CSL, JSL)  
- Are bulky and not suitable for daily use  

There is a need for a **lightweight, real-time ISL translation system**.

---

## 💡 Proposed Solution
A smart glove embedded with flex sensors detects finger movements and converts them into electrical signals.

These signals are processed using a microcontroller and translated into text displayed on an LCD screen.

---

## ⚙️ Components Used
- Arduino Nano  
- Flex Sensors (x5)  
- 10k Resistors  
- 16x2 LCD Display (I2C)  
- Li-ion Battery  
- TP4056 Charging Module  

---

## 🔄 Working Principle
1. Flex sensors detect finger bending  
2. Voltage changes are read by Arduino  
3. Sensor values are analyzed using logic conditions  
4. Specific gestures are mapped to words  
5. Output is displayed on LCD  

---

## ✨ Key Features
- Real-time gesture recognition  
- Portable and wearable design  
- Cost-effective solution  
- Supports Indian Sign Language (ISL)  
- Expandable to more gestures
